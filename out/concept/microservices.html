Microservices

Kiến trúc chia hệ thống thành các service nhỏ độc lập.

Mental model

Giống như một công ty chia thành nhiều phòng ban riêng biệt.

Core mechanism

  • Service độc lập
  • Giao tiếp qua API hoặc message queue

When to use

  • Hệ thống lớn, cần scale từng phần
  • Team nhiều, cần tách trách nhiệm
